## About
A ReactJS starter kit with Webpack, Babel 6, ES6, Hot Module Replacement, Linter and Sass.

## Installation
```bash
npm install
```

## Run Development Server
```bash
npm run start
```

## Build Production version
```bash
npm run build
```
